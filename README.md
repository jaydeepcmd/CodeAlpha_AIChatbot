# AI Chatbot - Java

## Description
This is a Java-based Artificial Intelligence chatbot developed as part of the CodeAlpha Internship.
The chatbot uses basic Natural Language Processing (NLP) techniques and rule-based logic to respond
to user queries interactively.

## Features
- Java console-based chatbot
- Basic NLP using keyword matching
- Rule-based AI responses
- Handles frequently asked questions (FAQs)
- Real-time user interaction

## Technologies Used
- Java
- Scanner
- NLP (basic text processing)

## How to Run
1. Compile the program:
   javac ChatBot.java
2. Run the program:
   java ChatBot

## Sample Questions
- Hello
- What is Java?
- Tell me about the internship
- Help
- Exit
